# BSTE static website

This is a static GitHub Pages-ready website. Upload all contents of this folder to the repository root (or publish this folder as the Pages source).

## Before publishing

1. Add the institute logo as `logo.png` beside `index.html`.
2. Put every result PDF inside `pdf/`, named exactly after its roll number, such as `pdf/90626.pdf`.
3. Keep `students-data.js` in the same folder. It contains the supplied 10,000 records and is used by both verification pages.

## Verification demo

Use Roll No. `90626` and Passing Year `2025` to test a known record.

The supplied records contain 10,000 student entries across 20 programme/course types and passing years 2022–2025.
